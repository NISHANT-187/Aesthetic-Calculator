# Aesthetic Calculator

A stylish web calculator built with HTML, CSS, and JavaScript. It has a glass-style interface, smooth animations, keyboard support, and a click sound for button presses.

## Features

- Clean and modern glassmorphism UI
- Smooth button hover, press, and ripple animations
- Click sound effect using the Web Audio API
- Sound on/off toggle
- Keyboard support for quick calculations
- Responsive layout for desktop and mobile screens
- Basic operations: addition, subtraction, multiplication, division, percentage, decimal values, clear, and delete

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Use

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Start calculating.

## Keyboard Shortcuts

- Numbers `0-9`: enter numbers
- `+`, `-`, `*`, `/`, `%`: operators
- `.`: decimal point
- `Enter` or `=`: calculate result
- `Backspace`: delete last character
- `Escape`: clear calculator
